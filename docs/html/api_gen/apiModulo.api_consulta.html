obterCamada(nome_tabela='', simples=False, campo='', filtro='')

Obtém todos os dados de uma tabela do tipo camada (tem suporte espacial). Não deve ser usado para dados de indicadores.

Parameters

  • nome_tabela – nome da tabela no banco

  • simples – quando True retorna apenas cod_camada e geometria

  • campo – quando filtro != ‘’, especifica qual campo adicional da tabela deve ser obtido

  • filtro – usado na cláusula where

Returns

Dados, na forma de geopandas

Return type

geopandas

obterTema(tema='', indexes='')

Obtem os dados na forma de um pandas/geopandas. Os Indicadores precisam compartilhar o mesmo suporte espacial

Parameters

  • tema – utilizado para filtrar os campos de tema, assunto e descrição

  • indexes – lista de ids de Indicadores no formato de string, Ex. [1, 2, 3, 4]

Returns

Dados obtidos no banco. Se houver campo espacial, geopandas. Se não, um pandas. Os metadados são também devolvidos.

Return type

geopandas, dict

obterTemaSoma(tema='', indexes='')

Obtém os dados dos indicadores e soma seus resultados. Garanta que todos os indicadores são do tipo numérico.

Parameters

  • tema – utilizado para filtrar os campos de tema, assunto e descrição

  • indexes – lista de ids de Indicadores no formato de string, Ex. [1, 2, 3, 4]

Returns

Dados obtidos no banco. Se houver campo espacial, geopandas. Se não, um pandas. Os metadados são também devolvidos.

Return type

geopandas, dict
