Constroi visualização em mapa de um indicador específico Obrigatório que o indicador tenha suporte espacial

Parameters

  • df – GeoDataframe carregado com os dados

  • meta – metadados das variáveis presentes no geodataframe

  • height – Altura de cada mapa. Por padrão 1200

  • width – Largura de cada mapa. Por padrão 600

  • MAPA_ZOOM – Zoom inicial do mapa. Por padrão 14.5
